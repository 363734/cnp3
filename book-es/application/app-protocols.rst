.. Application-level protocols

Protocolos de nivel de Aplicación
#################################


.. Many protocols have been defined for networked applications. In this section, we describe some of the important applications that are used on the Internet. We first explain the Domain Name System (DNS) that enables hosts to be identified by human-friendly names instead of the IPv4 or IPv6 addresses that are used by the network. Then, we describe the operation of electronic mail, one of the first killer applications on the global Internet, and the protocols used on world wide web. 

Se han definido muchos protocolos para aplicaciones de red. En esta sección describiremos algunas de las más importantes aplicaciones usadas en Internet. Primero explicaremos el Sistema de Nombres de Dominio (`Domain Name System`, DNS) que posibilita que los hosts sean identificados por nombres aceptables para los humanos, en lugar de las direcciones IPv4 o IPv6 que son usadas por la red. Luego describiremos la operación del correo electrónico, una de las primeras aplicaciones importantes de la Internet global, y los protocolos usados en la World Wide Web (`WWW`).
