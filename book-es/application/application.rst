=====================
La Capa de Aplicación
=====================

.. The Application Layer is the most important and most visible layer in computer networks. Applications reside in this layer and human users interact via those applications through the network. 

La Capa de Aplicación es la más importante y más visible en las redes de computadoras. Las aplicaciones residen en esta capa, y los usuarios humanos interactúan, a través de la red, mediante estas aplicaciones.

.. In this chapter, we first briefly describe the main principles of the application layer and focus on the two most important application models :  the client-server and the peer-to-peer models. Then, we review in detail two families of protocols that have proved to be very useful in the Internet : electronic mail and the protocols that allow access to information on the world wide web. We also describe the Domain Name System that allows humans to use user-friendly names while the hosts use 32 bits or 128 bits long IP addresses. 

En este capítulo, primero describiremos brevemente los principios básicos de la Capa de Aplicación, y nos dedicaremos a los dos modelos de aplicaciones más importantes: el modelo cliente-servidor y el modelo peer-to-peer. Luego revisaremos en detalle dos familias de protocolos que han demostrado ser muy útiles en Internet: el correo electrónico, y los protocolos que permiten acceder a la información en la World Wide Web (WWW). También describiremos el Sistema de Nombres de Dominio (`Domain Name System`, DNS), que permite usar nombres aceptables para los humanos, mientras que los nodos de la red usan direcciones IP de 32 o de 128 bits.
